# BugFixesWeapons

> Targeted fixes for weapon-side bugs and weapon-specific boon interactions.

Part of the [Speedrun modpack](https://github.com/h2pack-speedrun/speedrun-modpack).

## What It Does

BugFixesWeapons collects a set of fixes for weapon behavior and weapon-specific boon interactions that can behave incorrectly in live runs.

The current module includes fixes for:

- `Extra Dose Fix`
  Fixes Extra Dose interactions with Coat second punch and dash strike.
- `Aspect of Selene Fix`
  Skyfall is now properly registered as Hex when starting a new run, allowing Path of Stars to be offered directly without needing to pick a Selene's Gift first. Also Skyfall is always at full Moonglow
- `Axe and Blade Omega Channel Fix`
  Fixes Axe omega attack and Blade omega special not benefiting correctly from channeling bonus.
- `Tidal Ring Fix`
  Fixes Tidal Ring failing to hit the same mob twice with Circe Staff.

Use this module when you want weapon-side bug fixes applied without changing broader routing or utility behavior in the pack.

## Installation

Install via [r2modman](https://thunderstore.io/c/hades-ii/) or manually place in your `ReturnOfModding/plugins` folder.

This module is usually installed as part of the full [Speedrun modpack](https://github.com/h2pack-speedrun/speedrun-modpack), where it appears in the shared Speedrun UI with the other speedrun-focused modules.
